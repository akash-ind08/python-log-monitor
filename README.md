 Python Log Monitoring & Alert System

A lightweight Python tool that scans log files and detects critical errors like "ERROR" and "FAILED".

## 🚀 Features
- Detects error patterns in logs
- Configurable keywords
- Simple and fast execution
- Useful for basic monitoring automation

## 🛠 Tech Stack
- Python
- Linux (optional)

## 📂 Project Structure
python-log-monitor/
│── monitor.py
│── config.json
│── sample.log
│── requirements.txt

## ▶️ How to Run
1. Clone the repo
2. Run:
   python monitor.py

## 📌 Use Case
Helps developers and DevOps engineers quickly detect issues from logs and automate basic monitoring.
